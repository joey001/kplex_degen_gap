# kplex_degen_gap
The maximum k-plex algorithm parameterized by the degeneracy gap.

## Run the code

An example of computing the exact maximum 15-plex for the dataset soc-slashdot is as follows

```sh
$ ./KPLEX data/soc-slashdot.bin 15
```
